## Best Practices to Remember

- Always start scripts with `#!/bin/bash`
- Quote your variables (`"$var"`) to handle spaces safely
- Use `chmod +x` before running a new script
- Use meaningful variable and function names
- Test edge cases (empty input, negative numbers, missing files)
