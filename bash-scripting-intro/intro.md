# Introduction to Shell Scripting in Bash

Welcome! In this scenario you will learn the fundamentals of writing shell scripts using the **Bash** shell — one of the most widely used command-line interpreters on Linux and macOS.

## What You Will Learn

By the end of this scenario you will be able to:

- Create and run a Bash script
- Use variables to store and manipulate data
- Read input from the user
- Make decisions using conditional (`if`/`elif`/`else`) statements
- Repeat actions with `for` and `while` loops
- Organise reusable logic into functions

## Prerequisites

You only need a basic familiarity with the Linux command line (navigating directories, running commands). No prior scripting experience is required.

## Environment

A Ubuntu terminal is available on the right. All commands and scripts in this scenario can be run directly there.
