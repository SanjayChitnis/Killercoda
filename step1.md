# Step 1: The First Generation

To see an artifact in action, you need to provide a high-level descriptive prompt.

### Prompt to Try in Claude:
`Create an interactive React component that visualizes a Newton's Cradle. Use Tailwind CSS for styling and Framer Motion for animations.`

### What happens?
Claude will open a side-window (the Artifact window) and show you the working component immediately. You can interact with it, click the balls, and see the physics simulation.

Open [Claude.ai](https://claude.ai) and paste the prompt above. Notice how the code and the preview coexist.
