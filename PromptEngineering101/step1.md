# Zero-Shot vs Few-Shot Prompting

A **Zero-Shot** prompt is when you ask the AI to perform a task without any examples.
A **Few-Shot** prompt provides examples to guide the AI's output format and style.

### Step 1: Zero-Shot
Try asking an AI this:
`Translate the following sentence to French: "The AI Hub is amazing."`

### Step 2: Few-Shot (Better)
Try this instead:
```text
Task: Translate English to French.

Example 1:
English: "I love learning."
French: "J'adore apprendre."

Example 2:
English: "The weather is nice."
French: "Il fait beau."

Example 3:
English: "The AI Hub is amazing."
French: 
```

Notice how the examples set the pattern for the AI to follow.
